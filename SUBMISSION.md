Installation Instructions

Prerequisites

Node.js (v14 or above)

npm or yarn



Backend Setup

cd nest-api

npm install

npm run start:dev



Backend runs on: http://localhost:3000/



Frontend Setup

cd react-app

npm install

npm run dev



Frontend runs on: http://localhost:5173/
